# Datal0re Tap

## How do I install these formulae?

`brew install datal0re/tap/<formula>`

Or `brew tap datal0re/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "datal0re/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
